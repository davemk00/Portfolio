GIS - Helsinki City Bikes

2019/01/01

As a MSc GIS project I developed Helsinki Shared City bikes map. This included setting up a web server, SQL database, and backend funcationality on a Raspberry Pi.

Access: https://HelCityBikes.hopto.org/ This version includes feedback form, where the user can report bikes faults. The faults are visualised in the markers/popup.

If this site is not accessible, an alternative version is available at https://davemk00.github.io/HelCityBikes/ This version does not have SQL db as that database server is hosted elsewhere.
